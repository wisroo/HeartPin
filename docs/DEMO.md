# HeartPin Phase 0 데모 가이드 (2026-06-06)

## 1. 시작 전 체크리스트 (시연 30분 전)

- [ ] 외장 SSD 연결 확인 (`/Volumes/Extreme SSD/여가/상우♥︎`)
- [ ] 터미널에서 **`./scripts/demo.sh`** 실행 → 접속 주소 + **QR코드** 출력 (서버가 죽어도 자동 재시작)
- [ ] Mac 브라우저에서 `http://localhost:3300` 열어 지도 확인
- [ ] Mac 브라우저에서 `http://localhost:3300/api/status` 열어 `ok: true`, `vault.external: true`, `record_exists: true` 또는 첫 업로드 전 상태 확인
- [ ] 폰 2대를 **Mac과 같은 와이파이**에 연결 → 카메라로 QR 스캔해 접속 확인
- [ ] 폰에서 접속 시 Mac에 방화벽 팝업이 뜨면 **"허용"**
- [ ] (플랜 B) 카페·회사망 등에서 폰→Mac이 안 붙으면: **폰 핫스팟**을 켜고 Mac+나머지 폰을 거기 연결 → 스크립트 재시작 → 새 QR 사용

## 2. 시연 동선 (~7분)

| # | 장면 | 멘트 포인트 |
|---|---|---|
| ① | **Mac 큰 화면**: 홈 → "지도에서 보기" → 기존 기록(당일 여행·광교호수공원) 열기 | "갤러리에 흩어진 사진을 여정으로 정리해 주는 커플 기록 서비스" |
| ② | **폰 2대 동시에** QR 스캔 → 런치 화면 (바라♥뇽이 · **D+N 자동 계산**) | "각자 폰에서 자기 캐릭터로" |
| ③ | 각자 "오늘 사진 올리기" → 사진 선택 → 업로드(진행률) | "원본은 그 자리에서 외장하드로, 기록용 압축본은 자동 생성" |
| ④ | **한 장씩 확인 플로우**: 자동 인식된 날짜·장소 → 새 장소 이름 짓기 → 담기 | "GPS로 같은 장소끼리 자동으로 묶어요" |
| ⑤ | 완료 화면(스팟별 집계) → **Mac 지도에 3초 내 자동 반영** ✨ | 핵심 와우 포인트 — 두 폰의 사진이 한 지도에 |
| ⑥ | Mac에서 카드 덱 넘기기 + 말풍선 ✎ 편집 + 여행 이름 ✎ 수정 | "기록은 둘이 함께 다듬는 것" |
| ⑦ | (위치 없는 사진이 있으면) 정리함 → 배치 → **지도에서 위치 찍기** | 제약을 기능으로: "위치 없는 사진도 직접 핀 찍어 배치" |
| ⑧ | **"▶ 이 여정 재생하기"** — 풀스크린 시네마틱으로 마무리 + Finder로 SSD 폴더(원본·record.json) 열어 보여주기 | "기록은 클라우드 없이도 우리 하드에 통째로" |

## 3. 알려진 제약 — 질문 받았을 때

| 질문 | 답 |
|---|---|
| 갤럭시 사진은 왜 위치가 안 잡혀요? | 안드로이드 개인정보 정책이 브라우저에 위치 정보를 마스킹해서 전달함 (실측 확인). 아이폰은 정상. 갤럭시는 ①지도 찍기 ②카톡 원본→PC 업로드로 보완, **근본 해소는 네이티브 앱(Phase 3.5/4)** |
| 다른 장소에서도 돼요? | 폰 핫스팟이면 어디서든 동일 (서버가 새 주소 QR 출력) |
| 데이터는 어디 저장돼요? | 외장 SSD에 원본+압축본+기록(record.json). 클라우드 전환(Supabase)은 같은 스키마로 설계돼 있어 어댑터 교체만 하면 됨 |
| 같은 사진 두 번 올리면? | SHA-256 해시로 중복 감지 — "이미 올린 사진"으로 건너뜀 |

## 4. 트러블슈팅

| 증상 | 조치 |
|---|---|
| 폰에서 접속 안 됨 | ① 주소에 `http://` 명시 ② 같은 와이파이인지 ③ 폰 VPN 끄기 ④ 핫스팟 플랜 B |
| 서버 죽음 | demo.sh가 2초 내 자동 재시작 — 화면만 새로고침 |
| 보관소가 맞는지 헷갈림 | `http://localhost:3300/api/status`에서 `vault.path`와 `vault.external` 확인 |
| IP가 바뀜 (장소 이동) | demo.sh 재시작 → 새 QR 스캔 |
| 화면이 옛날 버전 | 폰 브라우저 새로고침 (당겨서 새로고침) |
| 지도 타일 안 보임 | 인터넷 필요 (핫스팟이면 LTE로 자동 해결) |

- 다음 구동: `./scripts/demo.sh` 한 줄
